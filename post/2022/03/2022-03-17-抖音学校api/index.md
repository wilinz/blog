## 学校列表：

```
https://aweme.snssdk.com/aweme/v2/school/list/?cursor=0&count=10
```

## 搜索学校

```
https://aweme.snssdk.com/aweme/v2/school/search?cursor=0&count=5&keyword=北京
```

## 附近学校

```
https://aweme.snssdk.com/aweme/v2/school/nearest?longitude=&latitude=
```

## 二级学院

```
https://aweme.snssdk.com/aweme/v2/school/colleges?school_name=北京大学
```
